Space shooter is the name of the game, however Shoot-em-up is the type of the game that was made.

Inspirations for game concept:
Space Invaders

The Shoot-Em-Up file has all classes that contribute to the gameplay

The Shoot-Em-Up-MonoGameTool is the tool used to manage the media elements within the game includes images/graphics and sound. Should this game be implemented at your end, please consider configuring Monogame Pipeline; http://www.monogame.net/downloads/

The Video is the demonstration of how the game is played, where it was created and basic rules and concept of the game for the player. 

Project was made using Microsoft Visual Studio, C#.
